# Pet Shop Pitoco

## Descrição

O Pet Shop Pitoco é uma plataforma web desenvolvida para apresentar produtos e serviços oferecidos pelo Pet Shop de forma clara, organizada e acessível.

O sistema permite a visualização de produtos, consulta de serviços, realização de agendamentos on-line e contato com o estabelecimento por meio de uma interface moderna e responsiva.

---

## Objetivo

Desenvolver uma plataforma web para um pet shop, para permitir a divulgação de produtos e serviços, além da realização de agendamentos on-line para o atendimento dos pets.

---

## Funcionalidades

### Página Inicial

- Apresentação do Pet Shop Pitoco;
- Menu de navegação entre as páginas;
- Carrossel de imagens utilizando Bootstrap;
- Exibição da data e horário em tempo real utilizando JavaScript;
- Resumo dos produtos disponíveis;
- Resumo dos serviços oferecidos;
- Link para o sistema de agendamento.

### Produtos

Visualização de produtos organizados em categorias:

#### Acessórios

- Cama para Pet;
- Brinquedo de Corda.

#### Rações Não Perecíveis

- Ração Premium Adultos;
- Ração para Filhotes.

#### Higiene e Limpeza

- Tapete Higiênico;
- Fralda para Pets.

Cada produto possui:

- Imagem;
- Descrição;
- Valor.

### Serviços

Visualização dos serviços oferecidos pelo petshop:

- Banho;
- Tosa;
- Tele-busca. 

Cada serviço apresenta:

- Descrição;
- Informações do atendimento;
- Botão para agendamento.

### Agendamento

Sistema de agendamento contendo:

#### Dados do Cliente

- Nome;
- CPF;
- Endereço;
- Sexo;
- Telefone;
- E-mail.

#### Dados do Pet

- Nome;
- Raça;
- Idade.

#### Serviço

- Banho;
- Tosa.

#### Forma de Atendimento

- Tele-busca;
- Entrega no local.

#### Agendamento

- Seleção de data;
- Seleção de horário.

#### Recursos JavaScript

- Confirmação do agendamento;
- Exibição do resumo do agendamento;
- Exibição de data e hora em tempo real.

### Contato

Página contendo:

- Telefone;
- E-mail;
- Endereço;
- Horário de funcionamento;
- Formulário para envio de mensagem.

---

## Tecnologias Utilizadas

- HTML5;

- Bootstrap 5;
- JavaScript.

## Acessibilidade

Foram implementados recursos básicos de acessibilidade para melhorar a experiência dos usuários, incluindo:

- Utilização do atributo alt nas imagens;
- Associação entre labels e campos dos formulários;
- Estrutura semântica das páginas utilizando header, nav, main e footer.
