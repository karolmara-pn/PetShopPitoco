# Pet Shop Pitoco

## Descrição

O Pet Shop Pitoco é uma plataforma web desenvolvida com o objetivo de apresentar produtos e serviços oferecidos pelo Pet Shop Pitoco.

O sistema permite a visualização de produtos organizados em categorias e dos serviços disponíveis para os clientes.

## Funcionalidades

- Página inicial com apresentação do pet shop.
- Navegação entre páginas por meio de links.
- Visualização de produtos da categoria Acessórios.
- Visualização de produtos da categoria Rações Não Perecíveis.
- Visualização de produtos da categoria Higiene e Limpeza.
- Exibição de imagens dos produtos.
- Exibição de descrições e valores dos produtos.
- Visualização dos serviços oferecidos.
- Exibição de descrições e valores dos serviços.
- Página de contato com informações do pet shop.

## Estrutura do Projeto

- index.html
- produtos.html
- servicos.html
- contato.html
- pasta imagens
